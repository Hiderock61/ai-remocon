# AIリモコン

AIに話しかける前の重さを減らすリモコン。

## これは何？

AIリモコンは、プロンプト集ではありません。

ChatGPT・Gemini・Copilot・Claudeなどに貼る「最初の一文」を作るための、会話起動リモコンです。

ユーザーが最初からうまく説明できなくても、ボタンを押すだけで、AIとの会話を始める入口を作ります。

AIリモコン内ではチャットしません。  
答えも出しません。  
会話は外部AIで行います。

AIリモコンの役割は、人間のぐちゃっとした状態を、AIに渡せる入口へ変換することです。

## 何ができる？

### 通常
場所や状況から、AIに相談する入口を作ります。

通常タブには、知りたいことを目的に合わせてピンポイントに調べる「調べたい」入口もあります。

例：
* 調べたい
* 家の中
* 電車・移動中
* どこか行きたい
* 会社・学校・作業場
* 店・買い物中
* 病院・役所・窓口
* SNS・ネット中

### 日記・記録
今日の出来事やモヤモヤを、AIに聞き取ってもらう入口を作ります。

例：
* 今日を整理
* 気持ちを整理
* 明日に渡す
* 生活をメモ

### 仕事・対人
本音のままだと社会と事故ることがある時に、AIに一度、言葉のエアバッグを巻いてもらう面です。

例：
* 断る
* 言い換える
* 言われたことを整理
* 返信する
* 確認する

### 変換
日常の違和感を、軽くする・寸劇にする・言語化するための面です。

例：
* 重い話の軽量化
* 日常の寸劇化
* 違和感の言語化
* 変な例えにする
* アイデアにする

## 使い方

1. 面を選ぶ
2. ボタンを押す
3. コピーする
4. ChatGPT・Gemini・Copilot・Claudeなどに貼る
5. AIとの会話を始める

## 方針

AIリモコンは、AIチャットアプリではありません。

* アプリ内でチャットしない
* 入力フォームを作らない
* 答えを出さない
* 最初に貼る文章だけ作る
* 会話は外部AIで行う
* プロンプト本文は初期表示しない
* 必要な人だけ「文言を見る・調整する」で確認できる

## プライバシーと構造

このアプリは、GitHub Pagesで動く静的HTMLアプリです。

* サーバーを持ちません
* アプリ内に入力内容を保存しません
* 外部AIに貼る文章を生成するだけです
* 実際の会話は、貼った先のAIサービス上で行われます
