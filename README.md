# AIリモコン v0.3

AIリモコンは、AIに貼るためのプロンプトをボタンで作る小さなWebアプリです。

AIを勉強するためのアプリではありません。  
スマホで毎回プロンプトを書くのが面倒な時に、ボタンを押して、コピーして、ChatGPT / Gemini / Copilot / Claude などに貼るためのリモコンです。

## v0.3 ハンディ版

トップ画面には、日常で使いやすい8つのボタンを置いています。

- 😵 なんか困った
- 🥒 なんか暇
- 🕵️ これほんと？
- 😲 何が起こったんだ
- 🤯 だからなんだ
- 🦉 教えてください
- 🧩 この悩み、どう扱う？
- 🗣️ 英語圏ならどう言う？

## 使い方

1. ボタンを押す
2. 表示された文章をコピーする
3. 外部AIに貼る
4. そのあと、自分の状況や言葉を送る
